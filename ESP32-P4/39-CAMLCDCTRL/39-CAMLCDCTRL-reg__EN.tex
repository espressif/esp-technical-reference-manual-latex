\\(R/W)

1: LCD\_PCLK is high. \\
(R/W)
\label{fielddesc:LCDCAMCLKEN}\item [LCD\_CAM\_CLK\_EN] Set this bit to force enable the clock for all configuration registers. Clock gate is not used. (R/W)
