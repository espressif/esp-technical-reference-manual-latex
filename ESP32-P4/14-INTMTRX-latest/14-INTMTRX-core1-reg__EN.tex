\regfield{CORE1\_\regindex{SOURCE}\_SRC\_IN\_SEC\_FLAG}{1}{7}{{0}}%
\regfield{CORE1\_\regindex{SOURCE}\_SRC\_PASS\_IN\_SEC}{1}{6}{{0}}%
\regfield{CORE1\_\regindex{SOURCE}\_MAP}{6}{0}{{0}}%
\reglabel{Reset}\regnewline%
\begin{regdesc}\begin{reglist}
\label{fielddesc:CORE1LPWDTINTMAP}\item [CORE1\_\regindex{SOURCE}\_MAP] When \hyperref[fielddesc:CORE1SOURCESRCINSECFLAG]{CORE1\_\regindex{SOURCE}\_SRC\_IN\_SEC\_FLAG} is set to 1, this register can only be written by CPU while in Machine Mode.

Configures this register to map the interrupt signal from the peripheral interrupt sources \regindex{Source} to one of CPU1’s peripheral interrupts. The valid configuration range is 16 \verb+~+ 47.\\
When set to 0 \verb+~+ 15, the corresponding peripheral interrupt source is disabled (recommended value: 0). \\

Other values are invalid. \\

For the list of interrupt sources \regindex{Source}, see Table \ref{tab:peripheral-interrupt-config-status-source}.
(R/W)

\label{fielddesc:CORE1SOURCESRCPASSINSEC}\item[CORE1\_\regindex{SOURCE}\_SRC\_PASS\_IN\_SEC] Write 1 to allow the interrupt signal from interrupt source \regindex{Source} to be delegated to CPU1’s Machine Mode interrupt. Writable only by the CPU in Machine Mode. (R/W)

\label{fielddesc:CORE1SOURCESRCINSECFLAG}\item
[CORE1\_\regindex{SOURCE}\_SRC\_IN\_SEC\_FLAG] Write 1 to configure the interrupt signal from interrupt source \regindex{Source} as a secure interrupt source (that is, its interrupt mapping can be configured only by the CPU running in Machine Mode). Writable only by the CPU in Machine Mode.
(R/W)

\end{reglist}\end{regdesc}
\end{register}
