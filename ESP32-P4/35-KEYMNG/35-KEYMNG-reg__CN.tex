\reglabel{Reset}\regnewline%
\begin{regdesc}\begin{reglist}
\label{fielddesc:KEYMNGUSEEFUSEKEY}\item [KEYMNG\_USE\_EFUSE\_KEY] 配置是否使用 eFuse 密钥，而不使用密钥管理器部署的密钥。\\
\begin{itemize}
    \item Bit[0]：配置 ECDSA\_DS 密钥使用 eFuse 密钥\\
        0：无效\\
        1：使用 eFuse 密钥，仅当 EFUSE\_FORCE\_USE\_KEY\_MANAGER\_KEY 中的 Bit[0] 为 0 时有效。\\
    \item Bit[1]：配置 flash 密钥使用 eFuse 密钥\\
        0：无效\\
        1：使用 eFuse 密钥，仅当 EFUSE\_FORCE\_USE\_KEY\_MANAGER\_KEY 中的 Bit[1] 为 0 时有效。\\
    \item Bit[2]：配置 HMAC 密钥使用 eFuse 密钥 \\
        0：无效\\
        1：使用 eFuse 密钥，仅当 EFUSE\_FORCE\_USE\_KEY\_MANAGER\_KEY 中的 Bit[2] 为 0 时有效。\\
    \item Bit[3]： 配置 RSA\_DS\_KEY 使用 eFuse 密钥 \\
        0：无效\\
        1：使用 eFuse 密钥，仅当 EFUSE\_FORCE\_USE\_KEY\_MANAGER\_KEY 中的 Bit[3] 为 0 时有效。\\
    \item Bit[4]：配置 PSRAM 密钥使用 eFuse 密钥 \\
        0：无效\\
        1：使用 eFuse 密钥，仅当 EFUSE\_FORCE\_USE\_KEY\_MANAGER\_KEY 中的 Bit[4] 为 0 时有效。\\
\end{itemize}
(R/W)
\label{fielddesc:KEYMNGRNDSWITCHCYCLE}\item [KEYMNG\_RND\_SWITCH\_CYCLE] 配置密钥管理器切换随机数的周期，单位为密钥管理器时钟周期。仅当 \hyperref[fielddesc:EFUSEKMRNDSWITCHCYCLE]{EFUSE\_KM\_RND\_SWITCH\_CYCLE} 为 0 时生效。\\
注意：此切换周期建议配置为 TRNG 模块的时钟周期所对应的密钥管理器时钟周期数。\\
(R/W)
\label{fielddesc:KEYMNGUSESWINITKEY}\item [KEYMNG\_USE\_SW\_INIT\_KEY] 配置使用软件配置的 \begin{math}sw\_init\_key\end{math} 而不是 \hyperref[fielddesc:EFUSEKMINITKEY]{EFUSE\_KM\_INIT\_KEY}，仅当 \hyperref[fielddesc:EFUSEFORCEDISABLESWINITKEY]{EFUSE\_FORCE\_DISABLE\_SW\_INIT\_KEY} 为 0 时有效。\\
0：使用 \hyperref[fielddesc:EFUSEKMINITKEY]{EFUSE\_KM\_INIT\_KEY} \\
1：使用软件写入的初始密钥，即 \begin{math}sw\_init\_key\end{math} \\
(R/W)
\item [见下页]
\end{reglist}\end{regdesc}
