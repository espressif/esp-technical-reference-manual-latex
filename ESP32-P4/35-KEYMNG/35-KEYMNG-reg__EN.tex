\label{fielddesc:KEYMNGUSEEFUSEKEY}\item [KEYMNG\_USE\_EFUSE\_KEY] Configures whether or not to use eFuse key instead of Key Manager deployed key.\\
\begin{itemize}
    \item Bit[0]: configures whether to use eFuse key for ECDSA\_DS key.\\
        0: No effect\\
        1: Use eFuse Key, valid only when bit[0] in EFUSE\_FORCE\_USE\_KEY\_MANAGER\_KEY is 0.\\
    \item Bit[1] configures whether to use eFuse key for flash key.\\
        0: No effect\\

    \item Bit[3] configures whether to use eFuse key for RSA\_DS\_KEY. \\
        0: No effect\\
        1: Use eFuse Key, valid only when bit[3] in EFUSE\_FORCE\_USE\_KEY\_MANAGER\_KEY is 0.\\
    \item Bit[4] configures whether to use eFuse key for PSRAM key. \\
        0: No effect\\
        1: Use eFuse Key, valid only when bit[4] in EFUSE\_FORCE\_USE\_KEY\_MANAGER\_KEY is 0.\\
\end{itemize}
(R/W)
\label{fielddesc:KEYMNGRNDSWITCHCYCLE}\item [KEYMNG\_RND\_SWITCH\_CYCLE] Configures cycles of the Key Manager to switch random numbers.\\
Measurement: Key Manager clock cycle.\\
This field is valid only when EFUSE\_KM\_RND\_SWITCH\_CYCLE is set to 0.\\

(R/W)
