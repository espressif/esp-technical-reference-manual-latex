\begin{tiplisting}
\vspace{-2em}
\begin{enumerate}
\item \label{JTAG footnote: 1}
    Available: the corresponding JTAG function is available. \\
    Unavailable: the corresponding JTAG function is not available.
\item \label{JTAG footnote: 2}
    x: do not care.
\item \label{JTAG footnote: 3}
    "Temporary disable JTAG"  means that if there are an even number of bits "1" in EFUSE\_SOFT\_DIS\_JTAG[2:0], the JTAG function is turned on (the corresponding value in the table is 1), otherwise it is turned off (the corresponding value in the table is 0). However, under certain special conditions of the HMAC Accelerator in \chipname{}, the JTAG function may be turned on even if there is an odd number of bits "1" in EFUSE\_SOFT\_DIS\_JTAG[2:0].
    For information on how HMAC affects JTAG functionality, please refer to Chapter \hyperref[mod:hmac]{HMAC Accelerator}.
\item \label{JTAG footnote: 4}
    Please refer to Chapter \hyperref[mod:efuse]{eFuse Controller} to get more information about eFuse.
\item \label{JTAG footnote: 5}
    Please refer to \hyperref[mod:bootctrl]{Chip Boot Control} to get more information about the strapping pin GPIO34.
\end{enumerate}
\end{tiplisting}
