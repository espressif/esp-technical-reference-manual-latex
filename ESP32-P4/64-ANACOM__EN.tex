\section{Introduction}

\chipname{} integrates two analog voltage comparators, i.e., analog voltage comparator \regindex{0} and analog voltage comparator \regindex{1}. These comparators rely on special pads that support voltage comparison functionality to monitor voltage changes on these pads. Each analog voltage comparator has two pads associated with it, for the main voltage and the reference voltage respectively. The voltage comparison result generated by the analog voltage comparator can be used as Event Task Matrix (ETM) events to drive ETM tasks of other peripherals or trigger interrupts.

\begin{tiplisting}
The information provided in this chapter applies to both analog voltage comparator \regindex{0} and analog voltage comparator \regindex{1}. Unless otherwise indicated, the analog voltage comparator in this chapter refer to both analog voltage comparator \regindex{0} and analog voltage comparator \regindex{1}.
\end{tiplisting}

\section{Feature List}
\label{sec:anacom-features}

The analog voltage comparator has the following features:

\begin{itemize}
    \item Voltage comparison
    \begin{itemize}
        \item Configurable voltage comparison mode
        \item Configurable reference voltage
    \end{itemize}
    \item Interrupt upon changes of voltage comparison result
    \item ETM event generation
\end{itemize}

\section{Architectural Overview}
\label{sec:anacom-arch-overview}

Figure \ref{fig:anacom-architecture} shows the structure of the analog voltage comparator.

\begin{figure}[H]
    \centering
    \includegraphics[width=0.6\textwidth]{\modulefiles/figures/anacom_arch_20231018__EN}
    \caption{Analog Voltage Comparator Architecture}
    \label{fig:anacom-architecture}
\end{figure}

As shown in the figure above, the analog voltage comparator consists of:

\begin{itemize}
    \item Two pad interfaces:
        \begin{itemize}
            \item PAD1: Main voltage interface 
            \item PAD0: External reference voltage interface
        \end{itemize}
    \item VREF: Internal reference voltage interface
    \item mode\_control: Register-controlled switch for selecting the reference voltage source
    \item COMP\_OUT: Comparator voltage comparison result, which is an on-chip internal signal that cannot be directly accessed externally
    \begin{itemize}
            \item If the main voltage is higher than the reference voltage, the COMP\_OUT signal is high.
            \item If the main voltage is lower than the reference voltage, the COMP\_OUT signal is low.
        \end{itemize}
\end{itemize}

\section{Functional Description}
\label{sec:anacom-func-descr}

The analog voltage comparator has the following functions:

\begin{itemize}

    \item {\bfseries Voltage comparison}
    
          The analog voltage comparator relies on a specialized pads that support voltage comparison. For more information about general-purpose pads, please refer to Chapter \ref{mod:iomuxgpio} \nameref{mod:iomuxgpio}. Table below shows the mapping between the PAD of analog voltage comparator \regindex{0} and GPIO pins. 
          
          \begin{longtable}{ | L{5.5cm} | L{2cm} | L{2cm} | }
          \caption{Mapping Between PAD and GPIO} \label{tab:anacom-pad-gpio-mapping}\\ \hline
          \rowcolor{lightgray}
          \textbf{Analog Voltage Comparator} & \textbf{PAD0}  &  \textbf{PAD1}\\ \hline
          Analog voltage comparator \regindex{0} & GPIO51  &  GPIO52  \\ \hline
          Analog voltage comparator \regindex{1} & GPIO53  & GPIO54   \\ \hline
          \end{longtable}
         
          The comparison mode and reference voltage are configurable. For detailed configuration procedures, see Section \ref{sec:anacom-prog-procedures} \nameref{sec:anacom-prog-procedures}.
         
          The voltage comparison results are output as the COMP\_OUT signal.

    \item {\bfseries Voltage comparison mode control}

          The analog voltage comparator compares the main voltage with the reference voltage, which can either be internal or external. There are two voltage comparison modes depending on the reference voltage selection:

          \begin{itemize}
            \item Comparing the main voltage with the external reference voltage.
            \item Comparing the main voltage with the internal reference voltage.
          \end{itemize}
        
          The voltage comparison mode can be configured through \hyperref[fielddesc:LPSYSREGMODECOMP0]{LPSYSREG\_MODE\_COMP\regindex{n}} (\regindex{n} = 0 \verb+~+ 1).

    \item {\bfseries Reference voltage configuration}

          The internal reference voltage range is 0 \verb+~+ (0.7 × VDD\_IO\_6) V with a step of 0.1 × VDD\_IO\_6 V, where VDD\_IO\_6 is the supply voltage of the analog voltage comparator which is equal to the power supply voltage of the chip (usually 3.3 V). The internal reference voltage value can be configured through \hyperref[fielddesc:LPSYSREGDREFCOMP0]{LPSYSREG\_DREF\_COMP\regindex{n}} (n\regindex{n} = 0 \verb+~+ 1).
        
          The external reference voltage is accessed directly from the pad with an input range of 0 \verb+~+ {0.7 x VDD\_IO\_6} V, same as the internal reference voltage, requiring no additional configuration.

    \item {\bfseries Voltage comparison interrupt processing}
        
          The voltage comparison result, i.e., the COMP\_OUT signal, can be either high or low. Whenever the output value changes, a corresponding interrupt signal is generated.

\end{itemize}

\section{Event Task Matrix Feature}
\label{sec:anacom-etm-func}

The analog voltage comparator on \chipname{} supports the Event Task Matrix (ETM) function, which allows analog voltage comparator ETM events to trigger any peripherals' ETM tasks. This section introduces the ETM events related to analog voltage comparator. For more information, please refer to Chapter \ref{mod:evntaskmatrix} \nameref{mod:evntaskmatrix}.

The analog voltage comparator can generate the following ETM events:

\begin{itemize}
    \item GPIO\_EVT\_ZERO\_DET\_POS\regindex{n}: Indicates that the COMP\_OUT signal changes from low level to high level, i.e., the main voltage changes from below to above the reference voltage.
    \item GPIO\_EVT\_ZERO\_DET\_NEG\regindex{n}: Indicates that the COMP\_OUT signal changes from high level to low level, i.e., the main voltage changes from higher to lower than the reference voltage.
\end{itemize}

The analog voltage comparator does not support any ETM tasks.

\section{Programming Procedures}
\label{sec:anacom-prog-procedures}

The programming procedure for the analog voltage comparator is as follows:

\begin{tiplisting}
For more information about the register fields mentioned in this section, refer to Chapter \ref{mod:sysreg} \nameref{mod:sysreg}.
\end{tiplisting}

\begin{enumerate}
    \item Enable the comparator by setting \hyperref[fielddesc:LPSYSREGXPDCOMP0]{LPSYSREG\_XPD\_COMP\regindex{n}} (\regindex{n} = 0 \verb+~+ 1) to 1.
    \item Disable normal digital pad functions (including IE, OE, WPU, and WPD) for PAD1, and if using the external reference voltage, also for PAD0. For detailed configuration, see Chapter \ref{mod:iomuxgpio} \nameref{mod:iomuxgpio}.
    \item Configure the comparison mode by setting \hyperref[fielddesc:LPSYSREGMODECOMP0]{LPSYSREG\_MODE\_COMP\regindex{n}} (\regindex{n} = 0 \verb+~+ 1):
        \begin{itemize}
            \item 0: Configures to compare the main voltage with the internal reference voltage;
            \item 1: Configures to compare the main voltage with the external reference voltage.
        \end{itemize}
    \item Configure the internal reference voltage through \hyperref[fielddesc:LPSYSREGDREFCOMP0]{LPSYSREG\_DREF\_COMP\regindex{n}} (\regindex{n}n = 0 \verb+~+ 1), which offers a voltage range of 0 \verb+~+ (0.7 × VDD\_IO\_6) V with a step of 0.1 × VDD\_IO\_6 V.
\end{enumerate}
