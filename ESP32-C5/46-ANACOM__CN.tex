%%% Introduction %%%

\section{概述}

\chipname{} 芯片集成了一个模拟电压比较器。模拟电压比较器依靠支持电压比较功能的特殊芯片焊盘 (PAD) 实现，用于监测 PAD 上的电压变化。模拟电压比较器有外部两个电压接入点，分为主电压和参考电压。模拟电压比较器输出的电压比较结果可以作为 ETM（事件任务矩阵）事件来驱动其他外设的 ETM 任务，也可以触发中断。


%%%%%%%%%%%%%%%%%%%%
%%% Feature List %%%
%%%%%%%%%%%%%%%%%%%%

\section{主要特性}
\label{sec:anacom-features}

模拟电压比较器具有以下主要特性：

\begin{itemize}
    \item 支持电压比较功能
    \begin{itemize}
        \item 电压比较模式可配置
        \item 内部参考电压值可配置
    \end{itemize}
    \item 支持电压比较中断
    \item 支持 ETM 事件
\end{itemize}

\section{系统架构}
\label{sec:anacom-arch-overview}

图 \ref{fig:anacom-architecture} 展示了模拟电压比较器的系统架构。

\begin{figure}[H]
    \centering
    \includegraphics[width=0.6\textwidth]{\modulefiles/figures/anacom_arch_20231018__CN}
    \caption{模拟电压比较器系统框图}
    \label{fig:anacom-architecture}
\end{figure}

如上图所示，模拟电压比较器的结构如下：

\begin{itemize}
    \item 两个 PAD 接口：
        \begin{itemize}
            \item PAD1：主电压接口
            \item PAD0：外部参考电压接口
        \end{itemize}
    \item VREF：内部参考电压接口
    \item mode\_control：寄存器控制的选择开关，用于选择参考电压的输入源
    \item COMP\_OUT：比较器输出的电压比较结果，为片上内部信号，外部无法直接获取
    \begin{itemize}
            \item 若主电压高于参考电压，COMP\_OUT 信号输出为高电平
            \item 若主电压低于参考电压，COMP\_OUT 信号输出为低电平
        \end{itemize}
\end{itemize}

\section{功能描述}
\label{sec:anacom-func-descr}

模拟电压比较器具有以下功能：

\begin{itemize}

    \item {\bfseries 电压比较}

          模拟电压比较器采用的是支持电压比较功能的特殊 PAD，这种特殊 PAD 由两个通用 PAD 组成。下表列出了模拟电压比较器的 PAD 与 GPIO 管脚之间的映射关系。

          \begin{longtable}{ | L{5cm} | L{2cm} | L{2cm} | }
          \caption{模拟电压比较器 PAD 和 GPIO 之间的映射关系} \label{tab:anacom-pad-gpio-mapping}\\ \hline
          \rowcolor{lightgray}
          \textbf{模拟电压比较器} & \textbf{PAD0}  &  \textbf{PAD1}\\ \hline
          模拟电压比较器 & GPIO8  &  GPIO9  \\ \hline

          \end{longtable}

          电压比较 PAD 的比较模式和参考电压可配置，具体配置流程见小节 \ref{sec:anacom-prog-procedures} \nameref{sec:anacom-prog-procedures}。

          电压比较结果输出为 COMP\_OUT 信号。

    \item {\bfseries 电压比较模式控制}

          模拟电压比较器将主电压与参考电压进行比较，参考电压可选内部参考电压和外部参考电压。根据参考电压的选择不同，电压比较模式分为两种：

          \begin{itemize}
            \item 主电压与外部参考电压比较；
            \item 主电压与内部参考电压比较；
          \end{itemize}

          电压比较模式可通过 \hyperref[regdesc:GPIOEXTPADCOMPCONFIG0REG]{GPIO\_EXT\_PAD\_COMP\_CONFIG\_0\_REG}  配置。

    \item {\bfseries 参考电压配置}

          内部参考电压范围为 0 \verb+~+ (0.7 × VDDPST1) V，步长为 0.1 × VDDPST1 V，其中 VDDPST1 为模拟电压比较器的供电电压，等于芯片的电源电压（通常为 3.3 V）。内部参考电压值可通过寄存器 \hyperref[fielddesc:GPIOEXTDREFCOMP0]{GPIO\_EXT\_DREF\_COMP\_0} 配置。

          外部参考电压直接从 PAD 接入，无需配置，输入范围与内部参考电压一致，为 0 \verb+~+ (0.7 x VDDPST1) V。

    \item {\bfseries 电压比较中断处理}

          电压比较的输出值 COMP\_OUT 信号为高电平或低电平，只要输出值发生变化，就会产生对应的中断信号。

\end{itemize}

\section{事件任务矩阵}
\label{sec:anacom-etm-func}

在 \chipname{} 中，模拟电压比较器支持 ETM 功能，即可以通过模拟电压比较器的 ETM 事件触发任意外设的 ETM 任务。关于 ETM 更多详细信息，请参考章节 \ref{mod:evntaskmatrix} \nameref{mod:evntaskmatrix}。这里仅介绍与模拟电压比较器相关的 ETM 事件。

模拟电压比较器可产生的 ETM 事件有：

\begin{itemize}
    \item GPIO\_EVT\_ZERO\_DET\_POS：表示 COMP\_OUT 信号由低电平变为高电平，即主电压从低于参考电压变为高于参考电压。
    \item GPIO\_EVT\_ZERO\_DET\_NEG：表示 COMP\_OUT 信号由高电平变为低电平，即主电压从高于参考电压变为低于参考电压。
\end{itemize}

模拟电压比较器不支持任何 ETM 任务。

\section{配置流程}
\label{sec:anacom-prog-procedures}

软件配置电压比较器功能的流程如下：

\begin{tiplisting}
有关本节提及的寄存器字段的更多信息，请参阅章节 \ref{mod:iomuxgpio} \textit{\nameref{mod:iomuxgpio}}。
\end{tiplisting}

\begin{enumerate}
    \item 将 \hyperref[fielddesc:PCRIOMUXFUNCCLKEN]{PCR\_IOMUX\_FUNC\_CLK\_EN} 设置为1。
    \item 将 \hyperref[fielddesc:GPIOEXTXPDCOMP0]{GPIO\_EXT\_XPD\_COMP\_0} 设置为 1，启用电压比较器功能。
    \item 禁用 PAD1 的正常数字 PAD 功能，包括 IE、OE、WPU 和 WPD，如果使用外部参考电压，则同时禁用 PAD0 的正常数字 PAD 功能。具体的配置方式请参考章节 \ref{mod:iomuxgpio} \nameref{mod:iomuxgpio}。
    \item 配置比较模式，将 \hyperref[regdesc:GPIOEXTPADCOMPCONFIG0REG]{GPIO\_EXT\_PAD\_COMP\_CONFIG\_0\_REG}  设置为：
        \begin{itemize}
            \item 0：选择主电压和内部参考电压进行比较；
            \item 1：选择主电压和外部参考电压进行比较。
        \end{itemize}
    \item 通过 \hyperref[fielddesc:GPIOEXTDREFCOMP0]{GPIO\_EXT\_DREF\_COMP\_0}  配置内部参考电压，可调电压范围为 0 \verb+~+ (0.7 × VDDPST1) V，步长为 0.1 × VDDPST1 V。
\end{enumerate}
