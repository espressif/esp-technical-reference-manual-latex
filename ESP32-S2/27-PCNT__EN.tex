\chapter{Pulse Count Controller (PCNT)}
\label{mod:pcnt}


The pulse count controller (PCNT) is designed to count input pulses and generate interrupts. It can increment or decrement a pulse counter value by keeping track of rising (positive) or falling (negative) edges of the input pulse signal. The PCNT has four independent pulse counters, called units which have their groups of registers. In this chapter, \regindex{n} denotes the number of a unit from 0 \verb+~+ 3.


Each unit includes two channels (ch0 and ch1) which can independently increment or decrement its pulse counter value. The remainder of the chapter will mostly focus on channel 0 (ch0) as the functionality of the two channels is identical.

As shown in Figure \ref{Figure:pcnt-block-diagram}, each channel has two input signals:
\begin{enumerate}
    \item One control signal (e.g. ctrl\_ch0\_u\regindex{n}, the control signal for ch0 of unit n)
    \item One input pulse signal (e.g. sig\_ch0\_u\regindex{n}, the input pulse signal for ch0 of unit n)
\end{enumerate}

\begin{figure}[H]
\includegraphics[width=0.4\textwidth]{\modulefiles/figures/pcnt-block-diagram.eps}
\centering
\caption{PCNT Block Diagram}
\label{Figure:pcnt-block-diagram}
\end{figure}

\section{Features}
A PCNT has the following features:
\begin{itemize}
    \item Four independent pulse counters (units)
    \item Each unit consists of two independent channels sharing one pulse counter
    \item All channels have input pulse signals (e.g. sig\_ch0\_u\regindex{n}) with their corresponding control signals (e.g. ctrl\_ch0\_u\regindex{n})
    \item Independent filtering of input pulse signals (sig\_ch0\_u\regindex{n} and sig\_ch1\_u\regindex{n}) and control signals (ctrl\_ch0\_u\regindex{n} and ctrl\_ch1\_u\regindex{n}) on each unit
    \item Each channel has the following parameters:
        \begin{enumerate}
            \item Selection between counting on positive or negative edges of the input pulse signal
            \item Configuration to Increment, Decrement, or Disable counter mode for control signal's high and low states
        \end{enumerate}
    \item Maximum frequency of pulses: 40 MHz
\end{itemize}


\section{Functional Description}\label{Subsection:Functional Description}

\begin{figure}[H]
\includegraphics[width=1\textwidth]{\modulefiles/figures/pcnt_arch_updated}
\centering
\caption{PCNT Unit Architecture}
\label{Figure:pulse_cnt_architecture}
\end{figure}

Figure \ref{Figure:pulse_cnt_architecture} shows PCNT's architecture. As stated above, ctrl\_ch0\_u\regindex{n} is the control signal for ch0 of unit n. Its high and low states can be assigned different counter modes and used for pulse counting of the channel's input pulse signal sig\_ch0\_u\regindex{n} on negative or positive edges. The available counter modes are as follows:

\begin{itemize}
\item Increment mode: When a channel detects an active edge of sig\_ch0\_u\regindex{n} (the one configured for counting), the counter value pulse\_cnt increases by 1. Upon reaching \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}}, pulse\_cnt is cleared. If the channel's counter mode is changed or if \hyperref[fielddesc:PCNTCNTPAUSEUN]{PCNT\_CNT\_PAUSE\_U\regindex{n}} is set before pulse\_cnt reaches \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}}, then pulse\_cnt freezes and its counter mode changes.

\item Decrement mode: When a channel detects an active edge of sig\_ch0\_u\regindex{n} (the one configured for counting), the counter value pulse\_cnt decreases by 1. Upon reaching \hyperref[fielddesc:PCNTCNTLLIMUN]{PCNT\_CNT\_L\_LIM\_U\regindex{n}}, pulse\_cnt is cleared. If the channel's counter mode is changed or if \hyperref[fielddesc:PCNTCNTPAUSEUN]{PCNT\_CNT\_PAUSE\_U\regindex{n}} is set before pulse\_cnt reaches \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}}, then pulse\_cnt freezes and its counter mode changes.

\item Disable mode: Counting is disabled, and the counter value pulse\_cnt freezes.
\end{itemize}

Each unit has one filter for all its control and input pulse signals. A filter can be enabled with the bit \hyperref[fielddesc:PCNTFILTERENUN]{PCNT\_FILTER\_EN\_U\regindex{n}}. The filter monitors the signals and ignores all the noise, i.e. the glitches with pulse widths shorter than \\\hyperref[fielddesc:PCNTFILTERTHRESUN]{PCNT\_FILTER\_THRES\_U\regindex{n}} APB clock cycles in length.

As previously mentioned, each unit has two channels which process different input pulse signals and increase or decrease values via their respective inc\_dec modules, then the two channels send these values to the adder module that is 16-bit wide with a sign bit. This adder can be suspended by setting \hyperref[fielddesc:PCNTCNTPAUSEUN]{PCNT\_CNT\_PAUSE\_U\regindex{n}}, and cleared by setting \hyperref[fielddesc:PCNTPULSECNTRSTUN]{PCNT\_PULSE\_CNT\_RST\_U\regindex{n}}.

The PCNT has five watchpoints that share one interrupt. The interrupt can be enabled or disabled by interrupt enable signals of each individual watchpoint.

\begin{itemize}
    \item Maximum count value: When pulse\_cnt reaches \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}}, an interrupt is triggered and \\\hyperref[fielddesc:PCNTCNTTHRHLIMLATUN]{PCNT\_CNT\_THR\_H\_LIM\_LAT\_U\regindex{n}} is high.
    \item Minimum count value: When pulse\_cnt reaches \hyperref[fielddesc:PCNTCNTLLIMUN]{PCNT\_CNT\_L\_LIM\_U\regindex{n}}, an interrupt is triggered and \\\hyperref[fielddesc:PCNTCNTTHRLLIMLATUN]{PCNT\_CNT\_THR\_L\_LIM\_LAT\_U\regindex{n}} is high.
    \item Two threshold values: When pulse\_cnt equals either \hyperref[fielddesc:PCNTCNTTHRES0UN]{PCNT\_CNT\_THRES0\_U\regindex{n}} or \hyperref[fielddesc:PCNTCNTTHRES1UN]{PCNT\_CNT\_THRES1\_U\regindex{n}}, an interrupt is triggered and either \hyperref[fielddesc:PCNTCNTTHRTHRES0LATUN]{PCNT\_CNT\_THR\_THRES0\_LAT\_U\regindex{n}} or \hyperref[fielddesc:PCNTCNTTHRTHRES1LATUN]{PCNT\_CNT\_THR\_THRES1\_LAT\_U\regindex{n}} is high respectively.
    \item Zero: When pulse\_cnt is 0, an interrupt is triggered and \hyperref[fielddesc:PCNTCNTTHRZEROLATUN]{PCNT\_CNT\_THR\_ZERO\_LAT\_U\regindex{n}} is valid.
\end{itemize}

\section{Applications}

In each unit, channel 0 and channel 1 can be configured to work independently or together. The three subsections below provide details of channel 0 incrementing independently, channel 0 decrementing independently, and channel 0 and channel 1 incrementing together. For other working modes not elaborated in this section (e.g. channel 1 incrementing/decremeting independently, or one channel incrementing while the other decrementing), reference can be made to these three subsections.

\subsection{Channel 0 Incrementing Independently} \label{Subsection:one channel inc}

\begin{figure}[h]
\fbox{\includegraphics[width=0.9\textwidth]{\modulefiles/figures/pulse_inc}}
\centering
\caption{Channel 0 Up Counting Diagram}
\label{Figure:pulse_add}
\end{figure}

Figure \ref{Figure:pulse_add} illustrates how channel 0 is configured to increment independently on the positive edge of sig\_ch0\_u\regindex{n} while channel 1 is disabled (see subsection \ref{Subsection:Functional Description} for how to disable channel 1). The configuration of channel 0 is shown below.

\begin{itemize}
    \item \hyperref[fielddesc:PCNTCH0LCTRLMODEUN]{PCNT\_CH0\_LCTRL\_MODE\_U\regindex{n}}=0: When ctrl\_ch0\_u\regindex{n} is low, the counter mode specified for the low state turns on, in this case it is Increment mode.
    \item \hyperref[fielddesc:PCNTCH0HCTRLMODEUN]{PCNT\_CH0\_HCTRL\_MODE\_U\regindex{n}}=2: When ctrl\_ch0\_u\regindex{n} is high, the counter mode specified for the low state turns on, in this case it is Disable mode.
    \item \hyperref[fielddesc:PCNTCH0POSMODEUN]{PCNT\_CH0\_POS\_MODE\_U\regindex{n}}=1: The counter increments on the positive edge of sig\_ch0\_u\regindex{n}.
    \item \hyperref[fielddesc:PCNTCH0NEGMODEUN]{PCNT\_CH0\_NEG\_MODE\_U\regindex{n}}=0: The counter idles on the negative edge of sig\_ch0\_u\regindex{n}.
    \item \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}}=5: When pulse\_cnt counts up to \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}}, it is cleared.
\end{itemize}


\subsection{Channel 0 Decrementing Independently} \label{Subsection:one channel dec}

\begin{figure}[h]
\fbox{\includegraphics[width=0.8\textwidth]{\modulefiles/figures/pulse_dec}}
\centering
\caption{Channel 0 Down Counting Diagram}
\label{Figure:pulse_dec}

Figure \ref{Figure:pulse_dec} illustrates how channel 0 is configured to decrement independently on the positive edge of sig\_ch0\_u\regindex{n} while channel 1 is disabled. The configuration of channel 0 in this case differs from that in Figure \ref{Figure:pulse_add} in the following aspects:

\begin{itemize}
    \item \hyperref[fielddesc:PCNTCH0POSMODEUN]{PCNT\_CH0\_POS\_MODE\_U\regindex{n}}=2: the counter decrements on the positive edge of sig\_ch0\_u\regindex{n}.
    \item \hyperref[fielddesc:PCNTCNTLLIMUN]{PCNT\_CNT\_L\_LIM\_U\regindex{n}}=-5: when pulse\_cnt counts down to \hyperref[fielddesc:PCNTCNTLLIMUN]{PCNT\_CNT\_L\_LIM\_U\regindex{n}}, it is cleared.
\end{itemize}

\subsection{Channel 0 and Channel 1 Incrementing Together}\label{Subsection:two channels inc}

\begin{figure}[h]
\fbox{\includegraphics[width=0.9\textwidth]{\modulefiles/figures/pulse_add_2}}
\centering
\caption{Two Channels Up Counting Diagram}
\label{Figure:pulse_add_2}
\end{figure}


Figure \ref{Figure:pulse_add_2} illustrates how channel 0 and channel 1 are configured to increment on the positive edge of sig\_ch0\_u\regindex{n} and sig\_ch1\_u\regindex{n} respectively at the same time. It can be seen in Figure \ref{Figure:pulse_add_2} that control signal ctrl\_ch0\_u\regindex{n} and ctrl\_ch1\_u\regindex{n} have the same waveform, so as input pulse signal sig\_ch0\_u\regindex{n} and sig\_ch1\_u\regindex{n}. The configuration procedure is shown below.

\begin{itemize}
    \item For channel 0:
    \begin{itemize}
        \item \hyperref[fielddesc:PCNTCH0LCTRLMODEUN]{PCNT\_CH0\_LCTRL\_MODE\_U\regindex{n}}=0: When ctrl\_ch0\_u\regindex{n} is low, the counter mode specified for the low state turns on, in this case it is Increment mode.

        \item \hyperref[fielddesc:PCNTCH0HCTRLMODEUN]{PCNT\_CH0\_HCTRL\_MODE\_U\regindex{n}}=2: When ctrl\_ch0\_u\regindex{n} is high, the counter mode specified for the low state turns on, in this case it is Disable mode.

        \item \hyperref[fielddesc:PCNTCH0POSMODEUN]{PCNT\_CH0\_POS\_MODE\_U\regindex{n}}=1: The counter increments on the positive edge of sig\_ch0\_u\regindex{n}.

        \item \hyperref[fielddesc:PCNTCH0NEGMODEUN]{PCNT\_CH0\_NEG\_MODE\_U\regindex{n}}=0: The counter idles on the negative edge of sig\_ch0\_u\regindex{n}.

    \end{itemize}

    \item For channel 1:
    \begin{itemize}
        \item \hyperref[fielddesc:PCNTCH1LCTRLMODEUN]{PCNT\_CH1\_LCTRL\_MODE\_U\regindex{n}}=0: When ctrl\_ch1\_u\regindex{n} is low, the counter mode specified for the low state turns on, in this case it is Increment mode.

        \item \hyperref[fielddesc:PCNTCH1HCTRLMODEUN]{PCNT\_CH1\_HCTRL\_MODE\_U\regindex{n}}=2: When ctrl\_ch1\_u\regindex{n} is high, the counter mode specified for the low state turns on, in this case it is Disable mode.

        \item \hyperref[fielddesc:PCNTCH1POSMODEUN]{PCNT\_CH1\_POS\_MODE\_U\regindex{n}}=1: The counter increments on the positive edge of sig\_ch1\_u\regindex{n}.

        \item \hyperref[fielddesc:PCNTCH1NEGMODEUN]{PCNT\_CH1\_NEG\_MODE\_U\regindex{n}}=0: The counter idles on the negative edge of sig\_ch1\_u\regindex{n}.

    \end{itemize}

    \item \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}}=10: When pulse\_cnt counts up to \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}}, it is cleared.

\end{itemize}
