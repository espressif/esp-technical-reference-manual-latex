\chapter{定时器组 (TIMG)}
\label{mod:timg}


\section{概述}
通用定时器可用于准确设定时间间隔、在一定间隔后触发中断（周期或非周期的）或充当硬件时钟。如图\ref{Figure:timergroup}所示，ESP32-S2包含两个定时器组，即定时器组 0 和定时器组 1。每个定时器组有两个通用定时器（下文用 T\regindex{x} 表示， \regindex{x} 为 0 或 1）和一个主系统看门狗定时器。所有通用定时器均基于 16 位预分频器和 64 位可自动重新加载向上／向下计数器。

\begin{figure}[h! ]
    \centering
    \includegraphics[width=0.6\textwidth]{\modulefiles/figures/timergroupb5}\label{TIMG0}
    \caption{定时器组}
    \label{Figure:timergroup}
\end{figure}

本章包含主系统看门狗定时器的寄存器描述，其功能描述请参阅章节 \ref{mod:wdt} \textit{看门狗定时器}。本章中“定时器”指代通用定时器。

定时器具有如下功能：
\begin{itemize}
    \item 16 位时钟预分频器，分频系数为 1-65536
    \item 64 位时基计数器可配置成递增或递减
    \item 可读取时基计数器的实时值
    \item 暂停和恢复时基计数器
    \item 可配置的报警产生机制
    \item 计数器值重新加载（报警时自动重新加载或软件控制的即时重新加载）
    \item 电平触发中断和边沿触发中断机制
\end{itemize}

\newpage
\section{功能描述}

\subsection{16 位预分频器与时钟选择}

每个定时器可通过配置寄存器 \hyperref[regdesc:TIMGTXCONFIGREG]{TIMG\_T\regindex{x}CONFIG\_REG} 的 \hyperref[fielddesc:TIMGTXUSEXTAL]{TIMG\_T\regindex{x}\_USE\_XTAL} 字段，选择 APB 时钟 (APB\_CLK) 或外部时钟 (XTAL\_CLK) 作为源时钟。源时钟经 16 位预分频器分频，产生时基计数器使用的时基计数器时钟 (TB\_CLK)。16 位预分频器可通过 \hyperref[fielddesc:TIMGTXDIVIDER]{TIMG\_T\regindex{x}\_DIVIDER} 器字段配置，选取从 1 到 65536 之间的任意值。注意，将 \hyperref[fielddesc:TIMGTXDIVIDER]{TIMG\_T\regindex{x}\_DIVIDER} 置 0 后，分频系数会变为 65536。
定时器必须关闭（即 \hyperref[fielddesc:TIMGTXEN]{TIMG\_T\regindex{x}\_EN} 必须清零），才能更改 16 位预分频器。在定时器使能时更改 16 位预分频器会造成不可预知的结果。

\subsection{64 位时基计数器}
64 位时基计数器基于 TB\_CLK，可通过 \hyperref[fielddesc:TIMGTXINCREASE]{TIMG\_T\regindex{x}\_INCREASE} 字段配置为递增或递减。时基计数器可通过置位或清零 \hyperref[fielddesc:TIMGTXEN]{TIMG\_T\regindex{x}\_EN} 字段使能或关闭。使能时，时基计数器的值会在每个 TB\_CLK 周期递增或递减。关闭时，时基计数器暂停计数。注意，\hyperref[fielddesc:TIMGTXEN]{TIMG\_T\regindex{x}\_EN} 置位后， \hyperref[fielddesc:TIMGTXINCREASE]{TIMG\_T\regindex{x}\_INCREASE}字段还可以更改，时基计数器可立即改变计数方向。

时基计数器 64 位定时器的当前值必须被锁入两个寄存器，才能被 CPU 读取（因为 CPU 为 32 位）。在 \\\hyperref[regdesc:TIMGTXUPDATEREG]{TIMG\_T\regindex{x}UPDATE\_REG} 上写任意值，64 位定时器的值可立即锁入寄存器 \hyperref[regdesc:TIMGTXLOREG]{TIMG\_T\regindex{x}LO\_REG} 和 \hyperref[regdesc:TIMGTXHIREG]{TIMG\_T\regindex{x}HI\_REG} ，两个寄存器分别锁存低 32 位和高 32 位。在 \hyperref[regdesc:TIMGTXUPDATEREG]{TIMG\_T\regindex{x}UPDATE\_REG} 写入新值之前，寄存器 \hyperref[regdesc:TIMGTXLOREG]{TIMG\_T\regindex{x}LO\_REG} 和 \hyperref[regdesc:TIMGTXHIREG]{TIMG\_T\regindex{x}HI\_REG} 将保持不变，以便 CPU 读值。

\subsection{报警产生}
配置后，定时器的当前值与报警值相同时可触发报警。报警会产生中断，（可选择）让定时器的当前值自动重新加载（详见第 \ref{Timer_Reload} 节）。

64 位报警值可在 \hyperref[regdesc:TIMGTXALARMLOREG]{TIMG\_T\regindex{x}ALARMLO\_REG} 和 \hyperref[regdesc:TIMGTXALARMHIREG]{TIMG\_T\regindex{x}ALARMHI\_REG} 配置，两者分别代表报警值的低 32 位和高 32 位。但是，只有置位 \hyperref[fielddesc:TIMGTXALARMEN]{TIMG\_T\regindex{x}\_ALARM\_EN} 字段使能报警功能后，配置的报警值才会生效。为解决报警使能“过晚”（即报警使能时，定时器的值已过报警值），定时器的当前值高于向上计数器或低于向下计数器的报警值时也会立即触发报警。

报警时，\hyperref[fielddesc:TIMGTXALARMEN]{TIMG\_T\regindex{x}\_ALARM\_EN} 字段自动清零，在置位 \hyperref[fielddesc:TIMGTXALARMEN]{TIMG\_T\regindex{x}\_ALARM\_EN} 前不会再次报警。

\subsection{定时器重新加载} \label{Timer_Reload}
定时器重新加载指将定时器的低 32 位和高 32 位分别更新为寄存器 \hyperref[fielddesc:TIMGTXLOADLO]{TIMG\_T\regindex{x}\_LOAD\_LO} 和 \hyperref[fielddesc:TIMGTXLOADHI]{TIMG\_T\regindex{x}\_LOAD\_HI} 存储的重新加载值。但是，把重新加载值写入 \hyperref[fielddesc:TIMGTXLOADLO]{TIMG\_T\regindex{x}\_LOAD\_LO} 和 \hyperref[fielddesc:TIMGTXLOADHI]{TIMG\_T\regindex{x}\_LOAD\_HI} 寄存器不会改变定时器的当前值。写入的重新加载值会被定时器忽视，直到重新加载事件被触发。重新加载事件可由软件即时重新加载或报警时自动重新加载触发。

CPU 在寄存器 \hyperref[regdesc:TIMGTXLOADREG]{TIMG\_T\regindex{x}LOAD\_REG} 写任意值会触发软件即时重新加载，定时器的当前值会立即改变。如置位 \hyperref[fielddesc:TIMGTXEN]{TIMG\_T\regindex{x}\_EN} ，定时器会继续从新数值开始递增或递减计数。如清零 \hyperref[fielddesc:TIMGTXEN]{TIMG\_T\regindex{x}\_EN} ，定时器将保持当前值，直至计数重新使能。

报警时自动重新加载功能可让定时器在报警时重新加载，从重新加载值开始继续递增或递减计数。该功能通常用于周期性报警时重置定时器的值。\hyperref[fielddesc:TIMGTXAUTORELOAD]{TIMG\_T\regindex{x}\_AUTORELOAD} 字段置 1 可以使能报警时自动重新加载。如未使能该功能，报警后定时器的值会在过报警值后继续递增或递减。
\subsection{中断}
每个定时器都有一组连接至 CPU 的中断线（针对边沿中断和电平中断）。因此，每组定时器共有 6 个中断，命名如下：
\begin{itemize}
\item \label{int:TIMERSWDTLEVELINT}TIMG\_WDT\_LEVEL\_INT：组内看门狗定时器的电平中断，在看门狗定时器中断阶段超时后产生
\item \label{int:TIMERSWDTEDGEINT}TIMG\_WDT\_EDGE\_INT：组内看门狗定时器的边沿中断，在看门狗定时器中断阶段超时后产生
\item \label{int:TIMERSTXLEVELINT}TIMG\_T\regindex{x}\_LEVEL\_INT：通用定时器的电平中断，在报警时产生
\item \label{int:TIMERSTXEDGEINT}TIMG\_T\regindex{x}\_EDGE\_INT：通用定时器的边沿中断，在报警时产生
\end{itemize}

中断在报警（或看门狗定时器阶段超时）时触发。报警（或阶段超时）后，电平中断线会被拉高，直至手动清除。边沿中断则会在报警（或阶段超时）后产生一个短脉冲。要使能定时器的电平中断或边沿中断，应分别置位 \hyperref[fielddesc:TIMGTXLEVELINTEN]{TIMG\_T\regindex{x}\_LEVEL\_INT\_EN} 或 \hyperref[fielddesc:TIMGTXEDGEINTEN]{TIMG\_T\regindex{x}\_EDGE\_INT\_EN} 位。

每个定时器组的中断由一组寄存器控制。组内的每个定时器在该组寄存器中都有对应的位：

\begin{itemize}

\item \hyperref[fielddesc:TIMGTXINTRAW]{TIMG\_T\regindex{x}\_INT\_RAW}：
报警时置 1。该位在写值到对应的 \hyperref[fielddesc:TIMGTXINTCLR]{TIMG\_T\regindex{x}\_INT\_CLR} 位后才会被清零。
\item \hyperref[fielddesc:TIMGWDTINTRAW]{TIMG\_WDT\_INT\_RAW}：阶段超时时置 1。该位在写值到对应的 \hyperref[fielddesc:TIMGWDTINTCLR]{TIMG\_WDT\_INT\_CLR} 位后才会被清零。
\item \hyperref[fielddesc:TIMGTXINTST]{TIMG\_T\regindex{x}\_INT\_ST}：反映每个定时器中断的状态，通过用 \hyperref[fielddesc:TIMGTXINTENA]{TIMG\_T\regindex{x}\_INT\_ENA} 屏蔽 \hyperref[fielddesc:TIMGTXINTRAW]{TIMG\_T\regindex{x}\_INT\_RAW} 位来生成。对于电平中断而言，\hyperref[fielddesc:TIMGTXINTST]{TIMG\_T\regindex{x}\_INT\_ST} 反映了 \hyperref[fielddesc:TIMGTXINTRAW]{TIMG\_T\regindex{x}\_INT\_RAW} 的电平。
\item \hyperref[fielddesc:TIMGWDTINTST]{TIMG\_WDT\_INT\_ST}：反映每个看门狗定时器中断的状态，通过用 \hyperref[fielddesc:TIMGWDTINTENA]{TIMG\_WDT\_INT\_ENA} 屏蔽 \\\hyperref[fielddesc:TIMGWDTINTRAW]{TIMG\_WDT\_INT\_RAW} 位来生成。对于电平中断而言，\hyperref[fielddesc:TIMGWDTINTST]{TIMG\_WDT\_INT\_ST} 反映了 \hyperref[fielddesc:TIMGWDTINTRAW]{TIMG\_WDT\_INT\_RAW} 的电平。
\item \hyperref[fielddesc:TIMGTXINTENA]{TIMG\_T\regindex{x}\_INT\_ENA}：用于使能或屏蔽组内定时器的中断状态位。
\item \hyperref[fielddesc:TIMGWDTINTENA]{TIMG\_WDT\_INT\_ENA}：用于使能或屏蔽组内看门狗定时器的中断状态位。
\item \hyperref[fielddesc:TIMGTXINTCLR]{TIMG\_T\regindex{x}\_INT\_CLR}：置 1 此位清除定时器中断，定时器对应的 \hyperref[fielddesc:TIMGTXINTRAW]{TIMG\_T\regindex{x}\_INT\_RAW} 和 \hyperref[fielddesc:TIMGTXINTST]{TIMG\_T\regindex{x}\_INT\_ST} 位会清零。注意，使用电平中断前，必须清除定时器中断。
\item \hyperref[fielddesc:TIMGWDTINTCLR]{TIMG\_WDT\_INT\_CLR}：置 1 此位清除定时器中断，看门狗定时器对应的 \hyperref[fielddesc:TIMGWDTINTRAW]{TIMG\_WDT\_INT\_RAW} 和\\\hyperref[fielddesc:TIMGWDTINTST]{TIMG\_WDT\_INT\_ST} 位会清零。注意，使用电平中断前，必须清除看门狗定时器中断。
\end{itemize}
\section{配置与使用}
\subsection{定时器用作简单时钟}\label{simple_clock}
\begin{enumerate}
    \item 配置时基计数器。
    \begin{itemize}
        \item 置位 \hyperref[fielddesc:TIMGTXUSEXTAL]{TIMG\_T\regindex{x}\_USE\_XTAL} 字段选择源时钟。
        \item 置位 \hyperref[fielddesc:TIMGTXDIVIDER]{TIMG\_T\regindex{x}\_DIVIDER} 配置 16 位预分频器。
        \item 置位或清除 \hyperref[fielddesc:TIMGTXINCREASE]{TIMG\_T\regindex{x}\_INCREASE} 配置定时器方向。
        \item 在 \hyperref[fielddesc:TIMGTXLOADLO]{TIMG\_T\regindex{x}\_LOAD\_LO} 和 \hyperref[fielddesc:TIMGTXLOADHI]{TIMG\_T\regindex{x}\_LOAD\_HI} 上写初始值设置定时器的初始值，然后在 \\\hyperref[regdesc:TIMGTXLOADREG]{TIMG\_T\regindex{x}LOAD\_REG} 上写任意值将初始值重新加载进定时器。
    \end{itemize}
    \item 置位 \hyperref[fielddesc:TIMGTXEN]{TIMG\_T\regindex{x}\_EN} 开启定时器。
    \item 获得定时器的当前值。
    \begin{itemize}
        \item 在 \hyperref[regdesc:TIMGTXUPDATEREG]{TIMG\_T\regindex{x}UPDATE\_REG} 上写任意值锁存定时器的当前值。
        \item 从 \hyperref[regdesc:TIMGTXLOREG]{TIMG\_T\regindex{x}LO\_REG} 和 \hyperref[regdesc:TIMGTXHIREG]{TIMG\_T\regindex{x}HI\_REG} 读取锁存的定时器值。
    \end{itemize}
\end{enumerate}

\subsection{定时器用于一次性报警}\label{Timer_Alarm}
\begin{enumerate}
    \item 按照第 \ref{simple_clock} 节的第 1 步配置时基计数器。
    \item 配置报警。
    \begin{itemize}
        \item 置位 \hyperref[regdesc:TIMGTXALARMLOREG]{TIMG\_T\regindex{x}ALARMLO\_REG} 和 \hyperref[regdesc:TIMGTXALARMHIREG]{TIMG\_T\regindex{x}ALARMHI\_REG} 配置报警值。
        \item 置位 \hyperref[fielddesc:TIMGTXLEVELINTEN]{TIMG\_T\regindex{x}\_LEVEL\_INT\_EN} 或 \hyperref[fielddesc:TIMGTXEDGEINTEN]{TIMG\_T\regindex{x}\_EDGE\_INT\_EN} 分别使能电平中断和边沿中断。
    \end{itemize}
    \item 清零 \hyperref[fielddesc:TIMGTXAUTORELOAD]{TIMG\_T\regindex{x}\_AUTORELOAD} 关闭自动重新加载。
    \item 置位 \hyperref[fielddesc:TIMGTXEN]{TIMG\_T\regindex{x}\_EN} 开启定时器。
    \item 处理报警中断。
    \begin{itemize}
        \item 置位定时器对应的 \hyperref[fielddesc:TIMGTXINTCLR]{TIMG\_T\regindex{x}\_INT\_CLR} 位清除中断。
        \item 清零 \hyperref[fielddesc:TIMGTXEN]{TIMG\_T\regindex{x}\_EN} 关闭定时器。
    \end{itemize}
\end{enumerate}

\subsection{定时器用于周期性报警}

\begin{enumerate}
    \item 按照第 \ref{simple_clock} 节的第 1 步配置时基计数器。
    \item 按照第 \ref{Timer_Alarm} 节的第 2 步配置报警。
    \item 置位 \hyperref[fielddesc:TIMGTXAUTORELOAD]{TIMG\_T\regindex{x}\_AUTORELOAD} 使能自动重新加载，将重新加载值写入 \hyperref[fielddesc:TIMGTXLOADLO]{TIMG\_T\regindex{x}\_LOAD\_LO} 和 \hyperref[fielddesc:TIMGTXLOADHI]{TIMG\_T\regindex{x}\_LOAD\_HI}。
    \item 置位 \hyperref[fielddesc:TIMGTXEN]{TIMG\_T\regindex{x}\_EN} 开启定时器。
    \item 处理报警中断 （每次报警时重复）。
    \begin{itemize}
        \item 置位定时器对应的 \hyperref[fielddesc:TIMGTXINTCLR]{TIMG\_T\regindex{x}\_INT\_CLR} 位清除中断。
        \item 如下一次报警需要新的报警值和重新加载值（即每次都有不同的报警间隔），则应根据需要重新配置 \hyperref[regdesc:TIMGTXALARMLOREG]{TIMG\_T\regindex{x}ALARMLO\_REG}、\hyperref[regdesc:TIMGTXALARMHIREG]{TIMG\_T\regindex{x}ALARMHI\_REG}、\hyperref[fielddesc:TIMGTXLOADLO]{TIMG\_T\regindex{x}\_LOAD\_LO} 和 \\\hyperref[fielddesc:TIMGTXLOADHI]{TIMG\_T\regindex{x}\_LOAD\_HI} 。否则，上述寄存器应保持不变。
        \item 置位 \hyperref[fielddesc:TIMGTXALARMEN]{TIMG\_T\regindex{x}\_ALARM\_EN} 重新使能报警。
    \end{itemize}
    \item （最后一次报警时）关闭定时器。
    \begin{itemize}
        \item 置位定时器对应的 \hyperref[fielddesc:TIMGTXINTCLR]{TIMG\_T\regindex{x}\_INT\_CLR} 位清除中断。
        \item 清零 \hyperref[fielddesc:TIMGTXEN]{TIMG\_T\regindex{x}\_EN} 关闭定时器。
    \end{itemize}
\end{enumerate}
