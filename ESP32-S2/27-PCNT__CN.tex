\chapter{脉冲计数控制器 (PCNT)}
\label{mod:pcnt}


脉冲计数控制器 (Pulse Count Controller, PCNT) 用于对输入脉冲计数和产生中断，通过记录输入脉冲信号的上升沿或下降沿进行递增或递减计数。PCNT 有四个称为“单元”的独立脉冲计数器，这些单元拥有自己的寄存器。下文描述中 \regindex{n} 表示单元编号 0\verb+~+ 3 。

每个单元有两个通道（ ch0 和 ch1），可以独立配置为递增或递减计数。两个通道功能相同，下文以通道 0 (ch0) 为例进行介绍。

如图 \ref{Figure:pcnt-block-diagram} 所示，每个通道有两个输入信号：
\begin{enumerate}
    \item 一个控制信号（如 ctrl\_ch0\_u\regindex{n} 为 ch0 的控制信号）
    \item 一个脉冲输入信号（如  sig\_ch0\_u\regindex{n} 为 ch0 的脉冲输入信号）
\end{enumerate}

\begin{figure}[h!]
\includegraphics[width=0.4\textwidth]{\modulefiles/figures/pcnt-block-diagram.eps}
\centering
\caption{PCNT 框图}
\label{Figure:pcnt-block-diagram}
\end{figure}

\section{特性}
PCNT 有如下特性：
\begin{itemize}
    \item 四个脉冲计数器（单元），各自独立工作
    \item 每个单元有两个独立的通道，共用一个脉冲计数器
    \item 所有通道均有输入脉冲信号（如 sig\_ch0\_u\regindex{n}）和相应的控制信号（如 ctrl\_ch0\_u\regindex{n}）
    \item 滤波器独立工作，过滤每个单元的输入脉冲信号（sig\_ch0\_u\regindex{n} 和 sig\_ch1\_u\regindex{n}）和控制信号（ctrl\_ch0\_u\regindex{n} 和 ctrl\_ch1\_u\regindex{n}）
    \item 每个通道参数如下：
        \begin{enumerate}
            \item 选择在输入脉冲信号的上升沿或下降沿计数
            \item 在控制信号为高电平或低电平时可将计数模式配置为递增、递减或停止计数。
        \end{enumerate}
    \item 最大脉冲频率：40 MHz
\end{itemize}

\section{功能描述}\label{Subsection:Functional Description}

\begin{figure}[h!]
\includegraphics[width=1\textwidth]{\modulefiles/figures/pcnt_arch_updated}
\centering
\caption{PCNT 单元基本架构图}
\label{Figure:pulse_cnt_architecture}
\end{figure}

图 \ref{Figure:pulse_cnt_architecture} 为 PCNT 单元的基本架构图。 如上所述， ctrl\_ch0\_u\regindex{n} 为 ch0 的控制信号，当控制信号 ctrl\_ch0\_u\regindex{n} 为高电平或低电平时，都可配置输入脉冲信号 sig\_ch0\_u\regindex{n} 在上升沿和下降沿的不同计数模式。可选计数模式如下：

\begin{itemize}
\item 递增模式：通道检测到 sig\_ch0\_u\regindex{n} 的有效边沿时，计数器的值 pulse\_cnt 加 1。pulse\_cnt 大于等于 \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}} 时被清零。如果在 pulse\_cnt 达到 \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}} 前， 该通道的计数模式改变或置位 \hyperref[fielddesc:PCNTCNTPAUSEUN]{PCNT\_CNT\_PAUSE\_U\regindex{n}}，则 pulse\_cnt 计数模式改变，且不被清零。

\item 递减模式：通道检测到 sig\_ch0\_u\regindex{n} 的有效边沿时，计数器的值 pulse\_cnt 减 1。pulse\_cnt 小于等于 \hyperref[fielddesc:PCNTCNTLLIMUN]{PCNT\_CNT\_L\_LIM\_U\regindex{n}} 时被清零。如果在 pulse\_cnt 达到 \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}} 前， 该通道的计数模式改变或置位 \hyperref[fielddesc:PCNTCNTPAUSEUN]{PCNT\_CNT\_PAUSE\_U\regindex{n}} ， 则 pulse\_cnt 计数模式改变，且不被清零。

\item 停止计数：计数停止，计数器的值 pulse\_cnt 保持不变。
\end{itemize}

每个单元均有一个滤波器，用于该单元的所有控制信号和输入脉冲信号。置位 \hyperref[fielddesc:PCNTFILTERENUN]{PCNT\_FILTER\_EN\_U\regindex{n}} 位使能滤波器。滤波器监测信号，滤除脉冲宽度小于 \hyperref[fielddesc:PCNTFILTERTHRESUN]{PCNT\_FILTER\_THRES\_U\regindex{n}} 个 APB 时钟周期的线路毛刺。

如前文所述，每个单元有通道 0 和通道 1两个通道，处理不同的输入脉冲信号，并通过各自的 inc\_dec 模块递增或递减计数值。之后，两个通道将计数值发送给 adder 模块。adder 模块是一个带符号位的 16 位加法器。软件可以通过置位 \hyperref[fielddesc:PCNTCNTPAUSEUN]{PCNT\_CNT\_PAUSE\_U\regindex{n}} 暂停 adder，也可以通过置位 \hyperref[fielddesc:PCNTPULSECNTRSTUN]{PCNT\_PULSE\_CNT\_RST\_U\regindex{n}} 清零 adder。

PCNT 可以设置五个观察点，五个观察点共用一个中断，可以通过每个观察点各自的中断使能信号开启或屏蔽中断。

\begin{itemize}
    \item 最大计数值:当 pulse\_cnt 大于等于 \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}} 时， 产生中断， 同时 \hyperref[fielddesc:PCNTCNTTHRHLIMLATUN]{PCNT\_CNT\_THR\_H\_LIM\_LAT\_U\regindex{n}} 为高。
    \item 最小计数值:当 pulse\_cnt 小于等于 \hyperref[fielddesc:PCNTCNTLLIMUN]{PCNT\_CNT\_L\_LIM\_U\regindex{n}} 时，产生中断， 同时 \hyperref[fielddesc:PCNTCNTTHRLLIMLATUN]{PCNT\_CNT\_THR\_L\_LIM\_LAT\_U\regindex{n}} 为高。
    \item 两个中间阈值:当 pulse\_cnt 等于 \hyperref[fielddesc:PCNTCNTTHRES0UN]{PCNT\_CNT\_THRES0\_U\regindex{n}} 或者 \hyperref[fielddesc:PCNTCNTTHRES1UN]{PCNT\_CNT\_THRES1\_U\regindex{n}} 时，产生中断， 同时 \hyperref[fielddesc:PCNTCNTTHRTHRES0LATUN]{PCNT\_CNT\_THR\_THRES0\_LAT\_U\regindex{n}} 或 \hyperref[fielddesc:PCNTCNTTHRTHRES1LATUN]{PCNT\_CNT\_THR\_THRES1\_LAT\_U\regindex{n}} 为高。
    \item 零: 当 pulse\_cnt 等于 0 时，产生中断， 同时 \hyperref[fielddesc:PCNTCNTTHRZEROLATUN]{PCNT\_CNT\_THR\_ZERO\_LAT\_U\regindex{n}} 有效。
\end{itemize}

\section{应用实例}

每个单元的通道 0 和通道 1 可配置为独立工作或一起工作。下文详细说明了通道 0 独自递增计数、通道 0 独自递减计数和两个通道一起递增计数的应用实例。本节中未详述的通道工作模式（如通道 1 递减或递减、双通道一增一减），可参考这三种模式。

\subsection{通道 0 独自递增计数} \label{Subsection:one channel inc}

\begin{figure}[ht]
\fbox{\includegraphics[width=0.8\textwidth]{\modulefiles/figures/pulse_inc}}
\centering
\caption{通道 0 递增计数图}
\label{Figure:pulse_add}
\end{figure}

图 \ref{Figure:pulse_add} 为通道 0 在 sig\_ch0\_u\regindex{n} 上升沿独立递增计数的示意图，此时通道 1 关闭（请参阅 \ref{Subsection:Functional Description} 一节查看如何关闭通道 1）。通道 0 的配置如下所示。

\begin{itemize}
    \item \hyperref[fielddesc:PCNTCH0LCTRLMODEUN]{PCNT\_CH0\_LCTRL\_MODE\_U\regindex{n}}=0：当 ctrl\_ch0\_u\regindex{n} 为低电平时，递增计数。
    \item \hyperref[fielddesc:PCNTCH0HCTRLMODEUN]{PCNT\_CH0\_HCTRL\_MODE\_U\regindex{n}}=2：当 ctrl\_ch0\_u\regindex{n} 为高电平时，停止计数。
    \item \hyperref[fielddesc:PCNTCH0POSMODEUN]{PCNT\_CH0\_POS\_MODE\_U\regindex{n}}=1：在 sig\_ch0\_u\regindex{n} 的上升沿递增计数。
    \item \hyperref[fielddesc:PCNTCH0NEGMODEUN]{PCNT\_CH0\_NEG\_MODE\_U\regindex{n}}=0：在 sig\_ch0\_u\regindex{n} 的下降沿不计数。
    \item \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}}=5：pulse\_cnt 的值递增至 \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}} 时被清零。
\end{itemize}

\subsection{通道 0 独自递减计数} \label{Subsection:one channel dec}

\begin{figure}[ht]
\fbox{\includegraphics[width=0.8\textwidth]{\modulefiles/figures/pulse_dec}}
\centering
\caption{通道 0 递减计数图}
\label{Figure:pulse_dec}
\end{figure}

图 \ref{Figure:pulse_dec} 为通道 0 在 sig\_ch0\_u\regindex{n} 上升沿独立递减计数的示意图，此时通道 1 关闭。此时通道 0 的配置与图 \ref{Figure:pulse_add} 相比有如下区别：

\begin{itemize}
    \item \hyperref[fielddesc:PCNTCH0POSMODEUN]{PCNT\_CH0\_POS\_MODE\_U\regindex{n}}=2：即在 sig\_ch0\_u\regindex{n} 的上升沿递减计数。
    \item \hyperref[fielddesc:PCNTCNTLLIMUN]{PCNT\_CNT\_L\_LIM\_U\regindex{n}}=-5：pulse\_cnt 的值递减到 \hyperref[fielddesc:PCNTCNTLLIMUN]{PCNT\_CNT\_L\_LIM\_U\regindex{n}} 时被清零。
\end{itemize}

\subsection{通道 0 和通道 1 同时递增计数}\label{Subsection:two channels inc}

\begin{figure}[ht]
\fbox{\includegraphics[width=0.9\textwidth]{\modulefiles/figures/pulse_add_2}}
\centering
\caption{双通道递增计数图}
\label{Figure:pulse_add_2}
\end{figure}

图 \ref{Figure:pulse_add_2} 为通道 0 和通道 1 在 sig\_ch0\_u\regindex{n} 和 sig\_ch1\_u\regindex{n} 上升沿一同递增计数的示意图。如图 \ref{Figure:pulse_add_2} 所示，控制信号 ctrl\_ch0\_u\regindex{n} 与 ctrl\_ch1\_u\regindex{n} 的波形一致，输入脉冲信号 sig\_ch0\_u\regindex{n} 和 sig\_ch1\_u\regindex{n} 波形一致。具体配置如下：

\begin{itemize}
    \item 通道 0：
    \begin{itemize}
        \item \hyperref[fielddesc:PCNTCH0LCTRLMODEUN]{PCNT\_CH0\_LCTRL\_MODE\_U\regindex{n}}=0：当 ctrl\_ch0\_u\regindex{n} 为低电平时，递增计数。
        \item \hyperref[fielddesc:PCNTCH0HCTRLMODEUN]{PCNT\_CH0\_HCTRL\_MODE\_U\regindex{n}}=2：当 ctrl\_ch0\_u\regindex{n} 为高电平时，停止计数。
        \item \hyperref[fielddesc:PCNTCH0POSMODEUN]{PCNT\_CH0\_POS\_MODE\_U\regindex{n}}=1：在 sig\_ch0\_u\regindex{n} 的上升沿递增计数。
        \item \hyperref[fielddesc:PCNTCH0NEGMODEUN]{PCNT\_CH0\_NEG\_MODE\_U\regindex{n}}=0：在 sig\_ch0\_u\regindex{n} 的下降沿不计数。

    \end{itemize}

    \item 通道 1：
    \begin{itemize}
        \item \hyperref[fielddesc:PCNTCH1LCTRLMODEUN]{PCNT\_CH1\_LCTRL\_MODE\_U\regindex{n}}=0：当 ctrl\_ch1\_u\regindex{n} 为低电平时，递增计数。

        \item \hyperref[fielddesc:PCNTCH1HCTRLMODEUN]{PCNT\_CH1\_HCTRL\_MODE\_U\regindex{n}}=2：当 ctrl\_ch1\_u\regindex{n} 为高电平时，停止计数。

        \item \hyperref[fielddesc:PCNTCH1POSMODEUN]{PCNT\_CH1\_POS\_MODE\_U\regindex{n}}=1：在 sig\_ch1\_u\regindex{n} 的上升沿递增计数。

        \item \hyperref[fielddesc:PCNTCH1NEGMODEUN]{PCNT\_CH1\_NEG\_MODE\_U\regindex{n}}=0：在 sig\_ch1\_u\regindex{n} 的下降沿不计数。

    \end{itemize}
    \item \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}}=10： pulse\_cnt 递增至 \hyperref[fielddesc:PCNTCNTHLIMUN]{PCNT\_CNT\_H\_LIM\_U\regindex{n}} 时被清零。

\end{itemize}
