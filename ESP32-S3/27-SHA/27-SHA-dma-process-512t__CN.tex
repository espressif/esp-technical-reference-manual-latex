\textbf{DMA-SHA 的具体工作流程（SHA-512/{\it{t}}）}

\begin{enumerate}
    \item 选择运算标准。
        \begin{itemize}
            \item 配置 \hyperref[regdesc:SHAMODEREG]{SHA\_MODE\_REG} 寄存器为 7 选择 SHA-512/{\it{t}} 运算标准。
        \end{itemize}
    \item 选择是否启用中断。请将 \hyperref[regdesc:SHAINTENAREG]{SHA\_INT\_ENA\_REG} 寄存器配置为 1 以启动中断。
    \item 计算哈希初始值。
        \begin{enumerate}
            \item 配置 t\_string 和 t\_length，并将其写入 \hyperref[regdesc:SHATSTRINGREG]{SHA\_T\_STRING\_REG} 和 \hyperref[regdesc:SHATLENGTHREG]{SHA\_T\_LENGTH\_REG} 寄存器。具体请见 \ref{sec:sha-set-initial-hash} 章节。
            \item 对 \hyperref[regdesc:SHASTARTREG]{SHA\_START\_REG} 寄存器置 1，启动 SHA 加速器的运算。
            \item 轮询寄存器 \hyperref[regdesc:SHABUSYREG]{SHA\_BUSY\_REG} 一直到读回的值为 0，代表 SHA 硬件加速器已完成哈希初始值的计算。
        \end{enumerate}
    \item 配置块个数。
        \begin{itemize}
            \item 将待加密数据的总块数 M 写入  \hyperref[regdesc:SHADMABLOCKNUMREG]{SHA\_DMA\_BLOCK\_NUM\_REG} 寄存器。
        \end{itemize}
    \item 开始 DMA-SHA 运算。
        \begin{itemize}
            \item 对 \hyperref[regdesc:SHADMACONTINUEREG]{SHA\_DMA\_CONTINUE\_REG} 置 1 启动 SHA 加速器。
        \end{itemize}
    \item 等待 DMA-SHA 运算结束。判断 DMA-SHA 运算结束有以下两种方法：
        \begin{itemize}
            \item 轮询寄存器 \hyperref[regdesc:SHABUSYREG]{SHA\_BUSY\_REG} 结果为 0。
            \item 等待中断信号产生。此时，应及时通过软件将 \hyperref[regdesc:SHAINTCLEARREG]{SHA\_INT\_CLEAR\_REG} 寄存器置为 1 以清除中断。
        \end{itemize} 
    \item 获取信息摘要
        \begin{itemize}
            \item 从寄存器堆 \hyperref[regdesc:SHAHNREG]{SHA\_H\_\regindex{n}\_REG} 取出信息摘要。
        \end{itemize}
\end{enumerate}
