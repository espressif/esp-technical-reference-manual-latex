\chapter{DPort 寄存器}
\label{mod:dportreg}

\section{概述}

ESP32 集成了丰富的外设，通过 DPort 寄存器控制时钟门控，功耗管理，外设以及系统核心模块的配置，可以使得系统在保持最佳性能的同时将功耗降到最低。系统通过 DPort 寄存器中的配置寄存器对各个模块进行配置。

\section{主要特性}
DPort 寄存器包含了多个对应外设和模块的寄存器：

\begin{itemize}
    \item 系统和存储器
    \item 复位和时钟
    \item 中断矩阵
    \item DMA
    \item MPU/MMU
    \item APP\_CPU 控制器
    \item 外设时钟门控和复位
\end{itemize}

\section{功能描述}

\subsection{系统和存储器寄存器}
系统和存储器寄存器（见表 \ref{sec:dport-reg-summ}）用于系统和存储器，例如 cache 配置和存储器映射。寄存器描述详见章节\textit{\nameref{mod:sysmem}}。


\subsection{复位和时钟寄存器}
复位和时钟寄存器（见表 \ref{sec:dport-reg-summ}）用于复位和时钟。寄存器描述详见章节\textit{\nameref{mod:resclk}}。


\subsection{中断矩阵寄存器}
中断矩阵寄存器（见表 \ref{sec:dport-reg-summ}）用于通过中断矩阵配置和映射中断。寄存器描述详见章节\textit{\nameref{mod:intmtrx}}。


\subsection{DMA 寄存器}
DMA 寄存器（见表 \ref{sec:dport-reg-summ}）用于 SPI DMA 配置。寄存器描述详见章节 \textit{\nameref{mod:dma}}。


\subsection{MPU/MMU 寄存器}
MPU/MMU 寄存器（见表 \ref{sec:dport-reg-summ}）用于 MPU/MMU 配置与操作控制。寄存器描述详见章节 \textit{\nameref{mod:mmumpu}}。


\subsection{APP\_CPU 控制器寄存器}
APP\_CPU 控制器寄存器（见表 \ref{sec:dport-reg-summ}）可用于 APP\_CPU 的基本配置，例如暂停任务的执行，以及设置从 ROM code 启动后的跳转地址。寄存器描述详见章节 \ref{sec:dport-reg}。需要注意的是，复位寄存器无法通过硬件清除。


\subsection{外设时钟门控和复位}\label{sec:peri-clock-gating-reset}

以下 DPort 寄存器用于控制外设时钟门控和复位，相应位分别为不同外设的门控使能和复位使能。寄存器描述详见章节 \ref{sec:dport-reg}。

\begin{itemize}
    \item \hyperref[regdesc:PERICLKENREG]{DPORT\_PERI\_CLK\_EN\_REG}
    \item \hyperref[regdesc:PERIRSTENREG]{DPORT\_PERI\_RST\_EN\_REG}
    \item \hyperref[regdesc:PERIPCLKENREG]{DPORT\_PERIP\_CLK\_EN\_REG}
    \item \hyperref[regdesc:PERIPRSTENREG]{DPORT\_PERIP\_RST\_EN\_REG}
    \item \hyperref[regdesc:WIFICLKENREG]{DPORT\_WIFI\_CLK\_EN\_REG}
    \item \hyperref[regdesc:WIFIRSTENREG]{DPORT\_WIFI\_RST\_EN\_REG}
\end{itemize}

\textbf{需要注意的是}：
\begin{itemize}
    \item 外设时钟门控和复位寄存器均为高电平有效。
    \item 复位寄存器无法通过硬件清除，因此软件将外设复位后需要清除复位寄存器。
    \item ESP32 具有低功耗特性，因此有些外设时钟默认为关闭状态。在启用这些外设之前，必须将外设的时钟打开，即把相应的 CLK\_EN 位置为 1，并且解除外设的复位状态，即把相应的 RST\_EN 位置为 0。

\end{itemize}
