\hypertarget{mcpwm}{}
\chapter{电机控制脉宽调制器 (MCPWM)}
\label{mod:mcpwm}

\section{概述}
电机控制脉宽调制器 (MCPWM) 外设用于电机和电源控制。它提供了六个 PWM 输出，可在几种拓扑结构中运行。常见的拓扑结构之一是用一对 PWM 输出来驱动 H 桥以控制电机旋转速度和旋转方向。

MCPWM 的时序和控制资源分为两种主要类型的模块：PWM 定时器和 PWM 操作器。每个 PWM 定时器提供定时参考，可以自由运行，或同步到其他定时器或外部源。每个 PWM 操作器具有用于为一个 PWM 通道生成波形对的所有控制资源。MCPWM 外设还包含专用捕获模块，用于需要精确定时外部事件的系统。

ESP32 有两个 MCPWM 外设，分别是 MCPWM0 和 MCPWM1。它们的控制寄存器分别位于从地址 0x3FF5E000 和 0x3FF6C000 开始的 4 KB 内存中。详见章节 \ref {section:register_summary} 寄存器列表。

\section{主要特性}
每个 MCPWM 外设都有一个时钟分频器（预分频器），三个 PWM 定时器，三个 PWM 操作器和一个捕获模块。图 \ref{figure:EPM_overview} 展示了 MCPWM 的模块和接口上的信号。PWM 定时器用于生成定时参考。PWM 操作器将根据定时参考生成所需的波形。通过配置，任一 PWM 操作器可以使用任一 PWM 定时器的定时参考。不同的 PWM 操作器可以使用相同的 PWM 定时器的定时参考来产生 PWM 信号。此外，不同的 PWM 操作器也可以使用不同的 PWM 定时器的值来生成单独的 PWM 信号。不同的 PWM 定时器也可进行同步。

\begin{figure}[H]
    \centering
    \includegraphics[width=0.9\textwidth]{./29-MCPWM/figures/pwm_outside.eps}
    \caption{MCPWM 外设概览}
    \label{figure:EPM_overview}
\end{figure}

以下是图 \ref{figure:EPM_overview} 中模块的功能概述：

\begin{itemize}
    \item PWM 定时器 0，1 和 2
        \begin{itemize}
            \item 每个 PWM 定时器都有一个专用的 8 位时钟预分频器。
            \item
            PWM 定时器中的 16 位计数的工作模式包括：递增计数模式，递减计数模式，递增递减循环计数模式。
            \item 硬件同步可以触发 PWM 定时器重载,重载值位于相位寄存器中；同时触发预分频的重启，从而同步定时器的时钟。同步源可以来自任何 GPIO 或任何其他 PWM 定时器的 sync\_out。
        \end{itemize}
    \item PWM 操作器 0，1 和 2
        \begin{itemize}
            \item 每个 PWM 操作器有两个 PWM 输出（PWM\textcolor{red}{\it{x}}A 和 PWM\textcolor{red}{\it{x}}B），可以在对称和非对称配置中独立工作。
            \item 软件异步优先控制 PWM 信号。
            \item 死区在上升沿和下降沿可配置，并可分别设置。
            \item 所有事件都可触发 CPU 中断。
            \item 通过高频载波信号调制 PWM 输出，在使用变压器隔离栅极驱动器时非常有用。
            \item 周期，时间戳寄存器和其他主要的控制寄存器有影子寄存器，更新方法灵活。
        \end{itemize}
    \item 故障检测模块
        \begin{itemize}
            \item 出现故障时，可选择在逐周期模式或一次性模式下处理。
            \item 故障条件可强制 PWM 输出高或低电平。
        \end{itemize}
    \item 捕获模块
        \begin{itemize}
            \item 旋转电机的速度测量。
            \item 位置传感器脉冲之间的间隔时间测量。
            \item 脉冲序列信号的周期和占空比测量。
            \item 从占空比编码的电流/电压传感器导出的解码电流或电压振幅。
            \item 3 个独立的捕获通道，各具备一个 32 位的时间戳寄存器。
            \item  输入捕获信号可以预分频，边沿极性可选。
            \item 捕获定时器可以与 PWM 定时器或外部信号同步。
            \item 3 个捕获通道上都可以产生中断。
        \end{itemize}
\end{itemize}

\subsection{PWM 定时器模块}
\label{section:timer_sub}
每个 MCPWM 外设都有三个 PWM 定时器模块。它们中的任何一个都可以决定三个 PWM 操作器模块中任意一个的必要事件时序。通过使用 GPIO 矩阵的同步信号，内置同步逻辑允许一个或多个 MCPWM 外设中的多个 PWM 定时器模块作为一个系统协同工作。

\subsubsection{PWM 定时器模块的配置}
 用户可配置 PWM 定时器模块的以下功能：

\begin{itemize}
    \item 通过指定 PWM 定时器频率或周期来控制事件发生的频率。
    \item 配置特定 PWM 定时器与其他 PWM 定时器或模块同步。
    \item 使 PWM 定时器与其他 PWM 定时器或模块同相。
    \item 设置定时器计数模式：递增，递减，或递增递减循环计数模式。
    \item 使用预分频器更改 PWM 定时器时钟（PT\_clk）的速率。 每个定时器都有自己的预分频器，通过寄存器 \hyperref[regdesc:PWMTIMER0CFG0REG]{PWM\_TIMER0\_CFG0\_REG} 的 PWM\_TIMER\textcolor{red}{\it{x}}\_PRESCALE 配置。PWM 定时器根据该寄存器的设置以较慢的速度递增或递减。
\end{itemize}

\subsubsection{PWM 定时器工作模式和定时事件生成}
 PWM 定时器有三种工作模式，由 PWM\textcolor{red}{\it{x}} 定时器模式寄存器配置：
\begin{itemize}
    \item 递增计数模式：\\
    定时器从零增加到周期寄存器中配置的值。一旦到达周期值，PWM 定时器清零，并再次开始递增。PWM 周期等于周期寄存器中的周期值 + 1。\\
    说明：周期寄存器为 PWM\_TIMER\textcolor{red}{\it{x}}\_PERIOD，\textcolor{red}{\it{x}} = 0, 1, 2，对应 \hyperref[regdesc:PWMTIMER0PERIOD]{PWM\_TIMER0\_PERIOD}，\hyperref[regdesc:PWMTIMER1PERIOD]{PWM\_TIMER1\_PERIOD}，\hyperref[regdesc:PWMTIMER2PERIOD]{PWM\_TIMER2\_PERIOD}。
    \item 递减计数模式：\\
    PWM 定时器从周期寄存器中的值开始递减到零。达到零后，将恢复为周期值，再次开始递减。在这种情况下，PWM 周期等于周期寄存器中的周期值 + 1。
    \item 递增-递减循环模式：\\
    此模式结合了上述两种模式。PWM 定时器从零开始递增，直到达到周期值，再次递减为零。PWM 定时器按照此模式循环递增递减。PWM 周期为周期寄存器的周期值 × 2 + 1。
\end{itemize}

\subsubsection{PWM 定时器影子寄存器}
\label{section:shdw_reg}
PWM 定时器周期寄存器和 PWM 定时器时钟预分频器寄存器具有影子寄存器。影子寄存器的作用是保存某个值，在与硬件同步的特定时刻写入有效寄存器。 两种寄存器类型定义如下：
\begin{itemize}
    \item 有效寄存器\\
     有效寄存器直接控制硬件执行的所有操作。
    \item 影子寄存器\\
    影子寄存器作为要写入有效寄存器的值的临时缓存。 在用户配置的某个时间点，影子寄存器中的值被写入有效寄存器。在此之前，影子寄存器的内容对受控硬件没有任何直接影响。 这有助于防止寄存器由软件异步修改时可能发生的错误硬件操作。影子寄存器和有效寄存器具有相同的存储器地址。 软件总是写入或读取影子寄存器。有效寄存器的更新时间点由其特定的更新方式寄存器决定。更新时间点可以是 PWM 定时器等于零时，PWM 定时器等于周期时，同步时间点或立即。软件可以触发强制全局更新，根据影子寄存器更新模块中的所有有效寄存器。
\end{itemize}

\subsubsection{PWM 定时器同步和锁相}
PWM 模块采用灵活的同步方法。每个 PWM 定时器都有一个同步输入和一个同步输出。同步输入可以从 GPIO 矩阵的三个同步输出和三个同步信号中选择。同步输出可以使用同步输入信号，或在 PWM 定时器等于周期或 PWM 定时器等于零时产生。因此，PWM 定时器可以通过同步使它们之间的相位锁定。在同步期间，PWM 定时器时钟预分频器将复位其计数器，以同步 PWM 定时器时钟。

\subsubsection{PWM 生成器模块}
\label{gen_submodule}
\subparagraph{PWM 生成器模块的作用}
此模块中生成或导入重要的时序事件，并转化为特定操作，在 PWM\textcolor{red}{\it{x}}A 和 PWM\textcolor{red}{\it{x}}B 输出处生成所需的波形。

PWM 生成器模块执行以下操作：
\begin{itemize}
    \item 基于使用寄存器 A 和 B 配置的时间戳生成定时事件。满足以下条件时发生定时事件：
    \begin{itemize}
        \item UTEA：PWM 定时器递增计数并且其值等于寄存器 A。
        \item UTEB：PWM 定时器递增计数并且其值等于寄存器 B。
        \item DTEA：PWM 定时器递减计数并且其值等于寄存器 A。
        \item DTEB：PWM 定时器递减计数并且其值等于寄存器 B。
    \end{itemize}
    \item 基于故障或同步事件生成 U/DT1，U/DT2 定时事件。
    \item 当这些定时事件同时发生时管理优先级。
    \item 基于定时事件产生置 1，置 0 和取反操作。
    \item 根据 PWM 生成器模块的配置来控制 PWM 占空比。
    \item 使用影子寄存器处理新的时间戳值，以防止 PWM 波形中的脉冲干扰。
\end{itemize}

\subparagraph{PWM 操作器影子寄存器}

\subparagraph{PWM 信号生成}
当某个定时事件发生时，PWM 生成器控制输出 PWM\textcolor{red}{\it{x}}A 和 PWM\textcolor{red}{\it{x}}B 的电平。定时事件通过 PWM 定时器计数方向（递增或递减）进一步限定。根据定时器计数方向，模块可以对 PWM 定时器递增或递减计数的阶段执行不同的操作。

可以在 PWM\textcolor{red}{\it{x}}A 和 PWM\textcolor{red}{\it{x}}B 输出上配置以下操作：

\begin{itemize}
    \item 置为高电平：\\ 将 PWM\textcolor{red}{\it{x}}A 或 PWM\textcolor{red}{\it{x}}B 的输出设置为高电平。
    \item 置为低电平：\\通过将 PWM\textcolor{red}{\it{x}}A 或 PWM\textcolor{red}{\it{x}}B 的输出设置为低电平来清除 PWM\textcolor{red}{\it{x}}A 或 PWM\textcolor{red}{\it{x}}B 的输出。
    \item 取反：\\将 PWM\textcolor{red}{\it{x}}A 或 PWM\textcolor{red}{\it{x}}B 的当前输出电平更改为相反的值。如果它当前被拉高，则拉低，或反之。
    \item 不进行操作：\\保持 PWM\textcolor{red}{\it{x}}A 和 PWM\textcolor{red}{\it{x}}B 输出电平不变。在这种状态下，仍然可以触发中断。
\end{itemize}

输出上的操作通过寄存器 \hyperref[regdesc:PWMGEN0AREG]{PWN\_GEN\textcolor{red}{\it{x}}\_A\_REG} 和 \hyperref[regdesc:PWMGEN0BREG]{PWN\_GEN\textcolor{red}{\it{x}}\_B\_REG} 配置。每一次输出的操作都独立配置。此外，基于事件在某个输出上灵活地执行不同的操作。表 \ref{table:gen_in} 中列举的任何事件都可以作用于 PWM\textcolor{red}{\it{x}}A 或 PWM\textcolor{red}{\it{x}}B 输出上。关于生成器 0，1 或 2 的寄存器信息，请参考第 \ref{section:register_summary} 章。

\subparagraph{死区生成器模块作用}
章节 \hyperref[gen_submodule]{PWM 生成器模块}讲述了在 PWM\textcolor{red}{\it{x}}A 和 PWM\textcolor{red}{\it{x}}B 上输出特定边沿位置的信号的几个配置选项。通过改变信号之间的边沿位置以及设置信号的占空比，可获得所需的死区。另一种方式是使用专门的死区生成器模块控制死区。

死区生成器模块的主要功能如下：
\begin{itemize}
    \item 根据单个 PWM\textcolor{red}{\it{x}}A 输入的死区生成信号对（PWM\textcolor{red}{\it{x}}A 和 PWM\textcolor{red}{\it{x}}B）

\subsubsection{PWM 载波模块}
将 PWM 输出耦合到电机驱动器可能需要使用变压器隔离。变压器只提供交流信号，而 PWM 信号的占空比可能在 0\% 到 100\% 之间变化。PWM 载波模块可以通过使用高频载波对其进行调制，将该信号传递给变压器。

\subparagraph{功能概述}
此模块的以下关键功能可配置：
\begin{itemize}
    \item 载波频率
    \item 第一个脉冲的脉宽
    \item 第二个以及之后的脉冲的占空比
    \item 开启/关闭载波
\end{itemize}

\subparagraph{操作要点}
PWM 载波时钟 (PC\_clk) 来自于 PWM\_clk。通过寄存器 \hyperref[regdesc:PWMCARRIER0CFGREG]{PWM\_CARRIER\textcolor{red}{\it{x}}\_CFG\_REG} 的 PWM\_CARRIER\textcolor{red}{\it{x}}\_PRESCALE 和 PWM\_CARRIER\textcolor{red}{\it{x}}\_DUTY 位配置频率和占空比。 一次性脉冲的功能在于提供高能量脉冲以接通电源开关。随后的脉冲用于保持上电的状态。一次性脉冲宽度可通过 PWM\_CARRIER\textcolor{red}{\it{x}}\_OSHTWTH 位进行配置。通过 PWM\_CARRIER\textcolor{red}{\it{x}}\_EN 位来使能／禁止载波模块。

\begin{center}

\end{center}

其中：
\begin{itemize}
    \item $T_{PMW\_clk}$ 为 PWM 时钟周期 (PWM\_clk)
    \item $(PWM\_CARRIER\textcolor{red}{\it{x}}\_OSHTWTH+1)$ 为一次性脉冲宽度值（取值范围：1-16）
    \item $(PWM\_CARRIER\textcolor{red}{\it{x}}\_PRESCALE+1)$ PWM 载波时钟 (PC\_clk) 预分频值
\end{itemize}

\begin{center}

\subsubsection{故障处理器模块}
 每个 MCPWM 外设都连接来自 GPIO 矩阵的 3 个故障信号（FAULT0，FAULT1 和 FAULT2）。这些信号用于指示外部故障状况，并且可由故障检测模块预处理后生成故障事件。故障事件通过执行用户代码，针对特定故障调整 MCPWM 输出。

\subparagraph{故障处理模块功能}
故障处理模块的主要功能为：
\begin{itemize}
    \item 在检测到故障时强制 PWM\textcolor{red}{\it{x}}A 和 PWM\textcolor{red}{\it{x}}B 输出信号进入以下状态之一：
    \begin{itemize}
        \item 高
        \item 低
        \item 取反
        \item 无
    \end{itemize}
    \item 在检测到过电流过载／短路时执行一次性跳闸（OST）。
    \item 逐周期跳闸（CBC）以提供限流操作。
    \item 每个故障信号单独分配一次性或逐周期操作。
    \item 每个故障输入都生成中断。
    \item 支持软件强制跳闸。
    \item 根据需要开启或关闭模块功能。
\end{itemize}

\subparagraph{操作与配置要点}
本节提供故障处理模块的操作要点和配置选项。

来自管脚的故障信号在 GPIO 矩阵中采样和同步。 为了保证故障脉冲采样的成功，每个脉冲持续时间必须至少为 2 个 APB 时钟周期。故障检测模块使用 PWM\_clk 对故障信号进行采样，因此来自 GPIO 矩阵的故障脉冲持续时间必须至少为 1 个 PWM\_clk 周期。因此，无论 APB 时钟周期和 PWM\_clk 周期的大小关系如何，管脚上的故障信号脉冲的宽度必须至少等于两个 APB 时钟周期与一个 PWM\_clk 周期的和。

 故障处理器模块可以使用故障信号 FAULT0 至 FAULT2 中的高电平或低电平来生成故障事件 fault\_event0 至 fault\_event2。 每个故障事件可以单独配置为进行 CBC 操作，OST 操作或无操作。

\begin{itemize}
    \item {\bfseries 逐周期 (CBC) 操作：}\\
    当 CBC 操作被触发，PWM\textcolor{red}{\it{x}}A 和 PWM\textcolor{red}{\it{x}}B 的状态立即根据寄存器 \hyperref[regdesc:PWMFH0CFG0REG]{PWM\_FH\textcolor{red}{\it{x}}\_A\_CBC\_U/D} 和 \\\hyperref[regdesc:PWMFH0CFG0REG]{PWM\_FH\textcolor{red}{\it{x}}\_B\_CBC\_U/D} 的设置改变。PWM 定时器递增或递减计数时，可指定不同的操作。不同的故障事件可触发不同的逐周期操作中断。通过状态寄存器 \hyperref[regdesc:PWMFH0STATUSREG]{PWM\_FH\textcolor{red}{\it{x}}\_CBC\_ON} 开启或关闭 CBC 操作。在没有故障事件时，将在指定时间点，即发生 D/UTEP 或 D/UTEZ 事件时清除 PWM\textcolor{red}{\it{x}}A/B 上的 CBC 操作。寄存器 \hyperref[regdesc:PWMFH0CFG1REG]{PWM\_FH\textcolor{red}{\it{x}}\_CBCPULSE} 控制决定 PWM\textcolor{red}{\it{x}}A 和 PWM\textcolor{red}{\it{x}}B 恢复正常的事件。因此，在此模式下，CBC 操作在每个 PWM 循环后清除或刷新。
    \item {\bfseries 一次性（OST）操作：}\\
	当 OST 操作被触发时，PWM\textcolor{red}{\it{x}}A 和 PWM\textcolor{red}{\it{x}}B 的状态立即根据寄存器 \hyperref[regdesc:PWMFH0CFG0REG]{PWM\_FH\textcolor{red}{\it{x}}\_A\_OST\_U/D} 和\\\hyperref[regdesc:PWMFH0CFG0REG]{PWM\_FH\textcolor{red}{\it{x}}\_B\_OST\_U/D} 改变。PWM 定时器递增或递减计数时，可配置不同的操作。不同的故障事件可触发不同的 OST 操作中断。通过状态寄存器  \hyperref[regdesc:PWMFH0STATUSREG]{PWM\_FH\textcolor{red}{\it{x}}\_OST\_ON} 开启或关闭 OST 操作。PWM\textcolor{red}{\it{x}}A/B 上的 OST 操作将在没有故障事件时不能自动清除。一次性操作必需人为通过将寄存器 \hyperref[regdesc:PWMFH0CFG0REG]{PWM\_FH\textcolor{red}{\it{x}}\_CLR\_OST} 的值取反来清除。
\end{itemize}

\subsection{捕获模块}
\subsubsection{介绍}

捕获模块包含 3 个完整的捕获通道。通道输入信号 CAP0，CAP1 和 CAP2 来自于 GPIO 矩阵。由于 GPIO 矩阵的灵活性，CAP0，CAP1 和 CAP2 可以通过任一管脚输入配置。 多个捕获通道可以来自同一管脚输入，而每个通道的预分频可以分别设置。此外，每个捕获通道还可以来自不同的管脚输入。因此，可以通过后台硬件用多种方式处理捕获信号，而不直接由 CPU 处理。

每个模块都有以下独立资源：
\begin{itemize}
    \item  一个 32 位定时器（计数器），可与 PWM 定时器，另一个模块或软件同步。
    \item 3 个捕获通道，每个通道配有一个 32 位时间戳和一个捕获预分频器。
    \item 任何捕获通道的边沿极性（上升/下降沿）可独立选择。
    \item 输入捕获信号预分频（分频取值范围：1 – 256）。
    \item 三个捕获事件都有中断功能。
\end{itemize}

\subsubsection{捕获定时器}
 捕获定时器是一个 32 位计数器，使能时不断递增计数。输入端的 APB 时钟频率通常为 80 MHz。发生同步事件时，加载计数器，其相位存储在寄存器 \hyperref[regdesc:PWMCAPTIMERPHASEREG]{PWM\_CAP\_TIMER\_PHASE\_REG} 中。同步事件可来自 PWM 定时器同步输出，PWM 模块同步输入，或软件。 该捕获定时器为所有 3 个捕获通道提供定时参考。

\subsubsection{捕获通道}

必要时，到达捕获通道的捕获信号可先被反相，然后预分频。最后，预处理后的捕获信号的指定边沿将触发捕获事件。在捕获事件发生时，捕获定时器的值存储在时间戳寄存器 \hyperref[regdesc:PWMCAPCH0REG]{PWM\_CAP\_CH\textcolor{red}{\it{x}}\_REG} 中。捕获事件中的不同捕获通道可生成不同的中断。触发捕获事件的边沿储存在寄存器 \hyperref[regdesc:PWMCAPSTATUSREG]{PWM\_CAP\textcolor{red}{\it{x}}\_EDGE} 中。捕获事件可由软件强制发生。
